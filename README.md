# React Background Changer

A simple React project that allows users to change the background color by clicking different color buttons. This project was built to practice React basics, state management, and event handling.

## Features

- Change background color with one click
- Multiple color options
- Responsive design
- Built with React and Vite
- Uses React `useState` Hook

## Technologies Used

- React.js
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3

## What I Learned

- React Functional Components
- useState Hook
- JSX
- Event Handling
- State Management
- Dynamic Styling
- Responsive Design

## Getting Started

Clone the repository:

```bash
git clone https://github.com/ManeelaKhan/React-Bg-Changer.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Open the local development server shown in the terminal.
